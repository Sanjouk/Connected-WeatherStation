Connected Weather Station & Smart Home Telemetry

This Flask application provides a real-time monitoring dashboard for our connected smart home sensor unit, delivering essential environmental data directly to you.

Real-Time Monitoring

Security Alerts

The integrated Passive Infrared (PIR) sensor monitors motion within the environment, instantly updating the status on the dashboard when movement is detected.

Robust Backend
